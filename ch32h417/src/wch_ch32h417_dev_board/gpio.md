# CH32H417 GPIO及复用功能（GPIO/AFIO）教程

本教程基于CH32H417系列芯片应用手册，详细讲解GPIO的基本特性、功能配置、复用机制及寄存器操作，帮助开发者快速掌握GPIO的使用方法，适用于嵌入式开发入门及芯片实操参考。

# 一、GPIO基础认知

## 1.1 GPIO概述

CH32H417芯片内置6组GPIO端口，分别为PA0~PA15、PB0~PB15、PC0~PC15、PD0~PD15、PE0~PE15、PF0~PF14，共95个GPIO引脚。多数引脚可通过软件配置为输入、输出或复用外设功能，是芯片与外部设备通信的核心接口。

## 1.2 引脚供电分类

不同GPIO引脚的供电电压不同，需根据实际应用选择合适的电源配置，具体分类如下：

- **VDD33供电**：PA9~PA12、PC13~PC15、PE3~PE6，额定3.3V；其中PC13~PC15在VDD33掉电时，自动切换为VBAT供电（适合低功耗场景）。

- **VDDIO供电**：PA0~PA8、PB2~PB7、PB15、PC4~PC5、PD8、PE2、PF6~PF10，额定3.3V，支持1.8V、2.5V、3.3V自适应电源。

- **VIO18供电（高速引脚）**：PA13~PA15、PB0~PB1、PB10~PB14等，内置I/O电压调节器，支持1.2V、1.8V、2.5V、3.3V动态切换，需通过PWR_CTLR寄存器[12:9]字段配置默认电压。

# 二、GPIO主要特征

每个GPIO引脚可配置为以下7种模式，覆盖绝大多数应用场景：

1. 浮空输入：引脚悬空，不接上下拉电阻，适用于外部信号输入（如按键、传感器）。

2. 开漏输出：仅N-MOS管工作，输出低电平有效，高电平需外部上拉电阻，适用于总线通信（如I2C）。

3. 上拉输入：引脚通过内部上拉电阻接VDD，默认电平为高，适用于稳定的输入信号检测。

4. 推挽输出：P-MOS和N-MOS管交替工作，可直接输出高/低电平，驱动能力强（如LED驱动）。

5. 下拉输入：引脚通过内部下拉电阻接VSS，默认电平为低，适用于低电平触发的信号输入。

6. 复用功能输入/输出：引脚作为芯片外设（如USART、SPI）的专用接口，由外设控制其输入/输出。

7. 模拟输入：禁用数字电路，引脚直接接入模拟外设（如ADC），用于采集模拟信号。

注：多数引脚支持复用功能，具体复用分配需参考芯片手册中“引脚复用表”，复用配置需配合AFIO寄存器操作。

# 三、GPIO功能详解

## 3.1 GPIO模块结构

GPIO内部结构主要分为3部分，核心逻辑如下：

- **保护二极管**：每个引脚内置两只保护二极管，防止静电或过压损坏芯片。

- **输入驱动模块**：包含弱上下拉电阻（可选）、TTL施密特触发器；模拟输入时禁用施密特触发器，数字输入时启用，将外部信号转换为芯片可识别的数字信号，存入输入数据寄存器。

- **输出驱动模块**：由一对P-MOS/N-MOS管组成，通过配置可实现推挽/开漏输出；输出信号由输出数据寄存器控制，或由复用外设直接控制。

## 3.2 初始化状态

芯片复位后，多数GPIO引脚默认配置为**浮空输入模式**；少数与HSE等外设相关的引脚，默认配置为外设复用模式（无需手动初始化）。具体初始化状态需参考芯片“引脚描述”章节。

## 3.3 外部中断功能

所有GPIO引脚均可配置为外部中断输入通道，核心规则如下：

- 外部中断通道序号与GPIO引脚位号必须一致（如PA1、PB1均对应EXTI1通道）。

- 一个外部中断通道最多只能映射一个GPIO引脚（一对一映射），需通过AFIO_EXTICR寄存器配置引脚与通道的映射关系。

## 3.4 复用功能与重映射

### 3.4.1 复用功能原理

GPIO引脚通过复用器连接到芯片内部外设，复用器一次仅允许一个外设的复用功能（AF0~AF15）连接到引脚，避免外设冲突。复位后，复用器默认选择AF0，可通过GPIOx_AFLR（低位）、GPIOx_AFHR（高位）寄存器配置复用功能序号。

### 3.4.2 复用配置规则

- 输入方向复用：端口配置为“复用输入模式”，上下拉电阻根据实际需求配置。

- 输出方向复用：端口配置为“复用输出模式”，推挽/开漏模式根据外设需求配置。

- 双向复用：端口配置为“复用输出模式”，驱动器自动切换为浮空输入模式。

### 3.4.3 功能重映射

当默认复用引脚被占用时，可通过重映射功能将外设复用引脚切换到其他空闲引脚，核心重映射场景如下：

- OSC32_IN/OSC32_OUT（LSE引脚）：LSEON=0时，可重映射为PC14/PC15（GPIO模式）；LSEON=1时，为LSE振荡器引脚。

- XI/XO（晶振引脚）：通过AFIO_PCFR1寄存器配置，可重映射为PD0/PD1（GPIO模式）。

- ADC、TIM2、SDMMC、UHSIF等外设：通过对应重映射位配置，具体参考手册“复用功能重映射”章节。

## 3.5 锁定机制

通过锁定机制可固定GPIO引脚的配置，防止误操作修改，操作流程如下：

1. 配置GPIOx_LCKR寄存器，设置需要锁定的引脚（LCK0~LCK15）。

2. 执行特定写序列：写1 → 写0 → 写1 → 读0 → 读1（最后一步可选，用于确认锁定生效）。

3. 锁定生效后，直至下次芯片复位，该引脚的配置（GPIOx_CFGLR/GPIOx_CFGHR）无法修改。

## 3.6 输入/输出/模拟模式配置详解

### 3.6.1 输入模式

配置为输入模式时，输出驱动断开，上下拉电阻可选，禁用复用功能和模拟输入；每个HB时钟周期会将引脚电平采样到输入数据寄存器（GPIOx_INDR），读取该寄存器可获取引脚实时电平。

### 3.6.2 输出模式

配置为输出模式时，输出驱动器启用（推挽/开漏），禁用上下拉电阻，启用TTL施密特触发器；引脚电平会被采样到输入数据寄存器，推挽模式下读取输出数据寄存器（GPIOx_OUTDR）可获取最后一次写入的值。

### 3.6.3 模拟输入模式

配置为模拟输入模式时，输出缓冲器断开，施密特触发器禁用，上下拉电阻禁用；输入数据寄存器读取值恒为0，引脚直接接入模拟外设（如ADC）。

## 3.7 常用外设GPIO配置推荐

不同外设的GPIO配置不同，以下为核心外设的推荐配置（完整配置参考手册表9-1至表9-26）：

|外设类型|引脚功能|推荐GPIO配置|
|---|---|---|
|USART（全双工）|TX、RX|TX：推挽复用输出；RX：浮空输入/上拉输入|
|SPI（主模式）|SCK、MOSI、MISO、NSS|SCK/MOSI：推挽复用输出；MISO：浮空输入；NSS：推挽复用输出|
|I2C|SCL、SDA|开漏复用输出|
|TIM（输出比较）|CHx|推挽复用输出|
|ADC/DAC|模拟输入/输出|模拟输入模式|
# 四、GPIO与AFIO寄存器操作

GPIO和AFIO的寄存器均需以32位（字）方式操作，核心寄存器分为GPIO寄存器和AFIO寄存器两类，以下讲解常用寄存器的功能与操作方法。

## 4.1 GPIO核心寄存器

### 4.1.1 配置寄存器（GPIOx_CFGLR/GPIOx_CFGHR）

用于配置GPIO引脚的模式（输入/输出/复用/模拟）和输出类型（推挽/开漏），分为低位（CFGLR，控制引脚0~7）和高位（CFGHR，控制引脚8~15）。

- **CNFy[1:0]（配置位）**：输入模式下控制输入类型，输出模式下控制输出类型和复用模式。

- **MODEy[1:0]（模式位）**：控制引脚模式，00=输入，01/10/11=输出（不同输出速度，需配合SPEED寄存器）。

复位值：0x44444444（默认浮空输入模式）。

### 4.1.2 输入/输出数据寄存器（GPIOx_INDR/GPIOx_OUTDR）

- GPIOx_INDR（只读）：位0~15对应引脚0~15的实时输入电平，读取该寄存器可获取引脚状态。

- GPIOx_OUTDR（读写）：位0~15对应引脚0~15的输出电平，写入该寄存器可控制引脚输出高/低。

复位值：INDR为0x0000XXXX（不确定值），OUTDR为0x00000000（默认低电平）。

### 4.1.3 置位/复位寄存器（GPIOx_BSHR/GPIOx_BCR）

- GPIOx_BSHR（只写）：位0~15（BSy）置1时，对应OUTDR位置1；位16~31（BRy）置1时，对应OUTDR位清0（BSy优先级高于BRy）。

- GPIOx_BCR（只写）：位0~15置1时，对应OUTDR位清0（与BSHR的BRy功能一致）。

复位值：均为0x00000000。

### 4.1.4 锁定寄存器（GPIOx_LCKR）

用于锁定GPIO配置，核心位为LCKK（锁定键）和LCKy（引脚锁定位），具体操作见3.5节“锁定机制”。

### 4.1.5 速度寄存器（GPIOx_SPEED）

控制GPIO输出速度，位0~31分为16组（SPEEDy[1:0]），对应引脚0~15，配置如下：

- 00：10MHz；01：50MHz；10：100MHz；11：180MHz

复位值：0x00000000（默认10MHz）。

## 4.2 AFIO核心寄存器

### 4.2.1 重映射寄存器1（AFIO_PCFR1）

核心用于配置外设功能重映射，关键位包括：

- PD0PD1_RM：控制PD0/PD1是否重映射为GPIO（1=GPIO，0=晶振引脚）。

- SDMMC_RM[1:0]：SDMMC外设引脚重映射配置。

- UHSIF_PORT_RM[1:0]、UHSIF_CLK_RM[1:0]：UHSIF外设引脚重映射配置。

- ADC1_ETRGINJ_RM/ADC1_ETRGREG_RM：ADC1外部触发重映射配置。

复位值：0x00000000。

### 4.2.2 复用功能寄存器（GPIOx_AFLR/GPIOx_AFHR）

用于配置GPIO引脚的复用功能序号（AF0~AF15），分为低位（AFLR，控制引脚0~7）和高位（AFHR，控制引脚8~15），每4位控制一个引脚（AFRy[3:0]）。

复位值：均为0x00000000（默认AF0）。

### 4.2.3 外部中断配置寄存器（AFIO_EXTICR1/AFIO_EXTICR2）

- AFIO_EXTICR1：控制EXTI0~EXTI7通道与GPIO引脚的映射。

- AFIO_EXTICR2：控制EXTI8~EXTI15通道与GPIO引脚的映射。

每4位控制一个中断通道（EXTIx[3:0]），配置值对应GPIO端口（0000=PA，0001=PB，以此类推）。

# 五、实操示例（以PA0配置为推挽输出为例）

以下为寄存器操作示例（基于C语言，寄存器地址参考手册），实现PA0引脚推挽输出高电平：

```c
#include "ch32h417.h"

void GPIO_PA0_Init(void)
{
    // 1. 使能GPIOA时钟（需先使能外设时钟，此处省略时钟配置）
    // 2. 配置PA0为推挽输出模式，速度50MHz
    R32_GPIOA_CFGLR &= ~(0x0F << 0);  // 清除PA0的配置位（CNF0[1:0]和MODE0[1:0]）
    R32_GPIOA_CFGLR |= (0x03 << 0);   // MODE0=11（输出模式），CNF0=00（推挽输出）
    R32_GPIOA_SPEED |= (0x01 << 0);   // SPEED0=01（50MHz）
    
    // 3. 控制PA0输出高电平
    R32_GPIOA_OUTDR |= (0x01 << 0);   // PA0置1
    // 或使用BSHR寄存器：R32_GPIOA_BSHR |= (0x01 << 0);
}

int main(void)
{
    GPIO_PA0_Init();
    while(1)
    {
        // 循环保持高电平，可添加延时实现高低电平切换
    }
}

```

# 六、注意事项

- GPIO引脚供电电压需与配置匹配，避免过压损坏芯片（如VDD33引脚不可接5V）。

- 复用功能配置时，需先配置GPIO模式为“复用输入/输出”，再配置AF寄存器选择复用功能序号。

- 外部中断配置时，需确保中断通道与引脚位号一致，且仅能映射一个GPIO引脚。

- 锁定机制生效后，需复位芯片才能重新配置GPIO，实操时需谨慎使用。

- 高速引脚（VIO18供电）的速度配置需结合PWR_CTLR寄存器，确保电压与速度匹配。

本教程仅涵盖GPIO核心功能与操作，详细的引脚复用分配、寄存器位定义及外设配合使用，需参考CH32H417系列应用手册。
